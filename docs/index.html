      Interpretation rules

      A finding is an indicator for investigation, not proof of misconduct. Compare hashes and package verification results against a trusted baseline, document chain of custody, and preserve the original host state.

      • Confirm whether the collector ran with sufficient privileges.
      • Review the detailed artifacts, not only the metric counts.
      • Compare with the same distribution, role, and approved configuration.
      • Verify REPORT-SHA256SUMS.txt before analysis.
      • Analyze a copy and protect the original report bundle.

      Privacy

      Keep generated evidence out of public repositories

      Reports can contain hostnames, account names, file paths, privilege relationships, and package deviations. Do not commit report bundles, attach them to public issues, or place them in the Pages directory.
